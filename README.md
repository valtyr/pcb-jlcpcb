# pcb-jlcpcb

JLCPCB parts library integration for [pcb](https://github.com/nickmass/pcb).

## Installation

### From crates.io

```bash
cargo install pcb-jlcpcb
```

### Using cargo-binstall

```bash
cargo binstall pcb-jlcpcb
```

### From source

```bash
git clone https://github.com/valtyr/pcb-jlcpcb
cd pcb-jlcpcb
cargo install --path .
```

## Usage

Search for parts in the JLCPCB basic parts library:

```bash
pcb-jlcpcb search "10k resistor"
```
